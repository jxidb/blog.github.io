hexo 部署 Github Page 踩坑

以此记录 hexo 部署 github page 时踩坑问题

插入七牛云外链图不显示问题

  1. 这是因为在 GitHub page 部署的网站是https协议的，而七牛云图片外链是http协议，
  2. 在 https 的网站中插入 http 协议的图片，会自动将 http 协议的图片转为 https 进行访问
  3. 而七牛云外链图片通过https协议是访问不了的
  4. 解决方法
    1. 换乘 https 协议的外联（但是七牛云的 https 要钱好像，所以我换了一个网站，也是免费的 https://thumbsnap.com/）
    2. 直接点击 Upload 选中需要上传的图片就可以

GitHub Page 问题

  1. 在新建仓库的时候，需要将仓库名设置为username.github.io，这样才能生成 page，
  2. 创建结束以后，直接在setting里面找到 page 就可以
  4. 点击后面的按钮，就可以访问网站，有时候可能会有延迟，需要等一会网页才能加载出来

路径问题

  1. 当 hexo 部署到 GitHub page 以后，网页可以正常访问，但是 css，js 等资源加载 404
  2. 是由于没有在 hexo 的配置文件中添加 root，
  4. 这个 root 后面跟的就是仓库名，前后两个/不能省略
